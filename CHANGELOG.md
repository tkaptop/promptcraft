# 更新日志 (Changelog)

## [0.6.3] - 2025-12-26

### 新增 ✨
- 新增“品牌概念单品 (Brand Concept Object)”模板，支持跨界产品设计的杂志大片提示词生成

## [0.6.2] - 2025-12-26

### 新增 ✨
- 新增“木质层叠艺术 (Wood Art & Xmas)”模板，支持激光切割风格的 3D 渲染提示词
- 新增圣诞主题词库 `xmas_theme`，支持多种圣诞元素自定义
- 新增系统语言自动检测功能，非中文系统环境默认开启英文界面

### 改进 🚀
- 优化了模板数据的版本管理机制
- 优化了移动端发现页的切换逻辑

## [0.4.1] - 2025-12-12

### 新增 ✨
- 导出图片添加模糊背景效果，提升视觉质感
- 导出底部水印区域增加版本号（动态获取）和网址二维码
- 移动端支持 Web Share API 保存图片到相册
- 导入时使用友好的 Toast 提示和自定义对话框
- 完整备份导入时显示详细数据统计信息

### 改进 🚀
- 导出图片格式从 PNG 改为 JPG（质量 92%），文件大小减少 60-70%
- 导出卡片重新布局：图片置顶（最高 300px）、标题简洁、版本号移至底部
- 导出按钮状态显示"导出中..."，防止重复点击
- 优化手机端顶部标题和按钮布局，减少拥挤感
- 移动端隐藏版本号和刷新按钮，优化图标尺寸
- 移除导入流程中的 `alert()` 和 `window.confirm()`

### 修复 🐛
- 修复图片 URL 失败后无法替换的问题
- 修复手机端导入模板"选择文件后没有动静"的问题
- 图片加载失败时显示 300×300 占位符

### 移除 ❌
- 导出图片底部移除 Prompt Template 标签，简化信息展示

## [0.4.0] - 2025-12-10

### 新增 ✨
- 模版列表支持瀑布流展示
- 新增标签过滤功能
- 支持导入/导出模版与词库（Beta）
- 新建模版支持上传本地图片或图片 URL
- 新增"经典场景微缩复刻"和"可视化企业成长之路"模版
- 补充公司、画幅比例、渲染风格（3D 像素）等词库

### 改进 🚀
- 默认使用本地存储保存模版与词库
- 长图导出等待图片加载并进行 Base64 预取，避免封面空白
- 支持一键刷新系统模版/词库并保留用户自定义数据
- `start.bat` 与启动脚本兼容性修复
- 浏览器自动打开逻辑优化
